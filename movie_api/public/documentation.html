MyFlix

The objective of this api is to focus on the server side componenets of a movies web application. The web application will provide information about movies, directors and different genres to users. Users will have the option to sign up with their personal information and create a list of their favorite movies.

Request Url Http Method Request Body Response Body
Get list of Movies /movies GET None Json objects show data on movies, description, genre, director, image
Get list of directors /movies[directors] GET None Json objects showing data on directors bio, birth date, death
Add New movies /movies Post Json object about movie being added Json object with the movie that was added and an id
Delete movies /movies Delete None message indicating the status
Add a New User /user Post Json about new username Json with information added and id
Delete a user /user Delete None Message indicating the status
